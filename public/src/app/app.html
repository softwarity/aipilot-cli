CLI Usage

Basic usage

aipilot-cli --agent claude

Options

aipilot-cli [options]

--agent <name>      # Agent: claude, gemini, codex
--agent ?           # Force agent selection
--workdir <path>    # Working directory
--agents            # List available agents
--sessions          # List saved sessions
--clear-session     # Delete current session
--clear-sessions    # Delete all sessions
--unpair <id>       # Remove paired mobile
--status            # Show PC and mobile status
--version           # Show version

Commands

/qr                 # Display pairing QR code
